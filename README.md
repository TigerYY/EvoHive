# EvoHive 智巢

**Official site 官网:** [https://evohive.net](https://evohive.net)

EvoHive helps small teams compose agents, share capabilities, govern knowledge, and evolve safely over time.

---

## 中文

**EvoHive（智巢）** 是面向 3–20 人扁平小团队的**团队智能体工作台**。让团队在隔离的工作空间中装配 Agent、共享能力、沉淀知识，并通过能力交换实现团队与团队之间的连接与复用。

- **不是**单一聊天助手，**是**团队级智能体运行环境与能力共享平台。
- 完整保留六元组（出入口、路由、频道、技能、记忆、沙箱），支持受监督的自进化。
- 支持私有部署、客户云主机、多租户 SaaS、混合部署。

**官网与产品介绍：** [https://evohive.net](https://evohive.net)

## English

**EvoHive (智巢)** is a **team agent workspace** for flat teams of 3–20 people. Teams compose agents, share capabilities, and govern knowledge in isolated workspaces, and connect with other teams through a capability exchange.

- **Not** a single chatbot — **is** a team-level agent runtime and capability-sharing platform.
- Preserves the six-tuple (Entry, Routing, Channels, Skills, Memory, Sandbox) and supports supervised self-evolution.
- Supports private deploy, customer cloud VM, multi-tenant SaaS, and hybrid deployment.

**Official website:** [https://evohive.net](https://evohive.net)

- `website/`: Static marketing site (one-page, zh/en toggle, responsive). Deploy to any static host or Nginx.
